## 📌 Description

This repository contains four Python programs that demonstrate **Object‑Oriented Programming (OOP)** and **file handling** in Python. Each program reads from or writes to text files, uses classes and methods, and includes automatic sample file creation if input files are missing.

## 🚀 How to Run

### Prerequisites
- Python 3.6 or higher installed
- Terminal / Command Prompt

## Run each program from its own folder

## P1 – Even / Odd Separator

Reads 20 integers from numbers.txt (auto‑created if missing)
Writes even numbers to even.txt and odd numbers to odd.txt

## P2 – Highest GWA Finder

Reads student records from students.txt (auto‑created with 20 sample entries)
Prints the student with the highest GWA (lowest numeric value)

## P3 – Multi‑line File Writer

Interactively asks for lines of text
Saves all lines to mylife.txt

## P4 – Square of Evens / Cube of Odds

Reads 20 integers from integers.txt (auto‑created if missing)
Writes squares of even numbers to double.txt
Writes cubes of odd numbers to triple.txt

## OOP Features
Each program is implemented as a class with methods.

Attributes use snake_case for readability.

Methods are focused on single responsibilities (e.g., read_numbers, separate, process).

Automatic sample file creation handles missing input files gracefully.
